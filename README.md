# 项目简介

## 核心功能
- 用户管理
  - 注册/登录：支持学号、密码注册和登录。
  - 个人信息管理：学生可以更新个人信息，如姓名、性别、联系方式等。
  - 权限管理：区分普通学生和系统管理员权限。
- 宿舍管理
  - 宿舍信息：展示所有宿舍的基本信息，包括位置、容量、设施等。
  - 宿舍分配：根据学生的个人信息和偏好，使用协同过滤算法推荐合适的宿舍。
  - 宿舍调整：支持学生申请调整宿舍，管理员审核调整请求。
- 入住管理
  - 入住记录：记录学生的入住信息，包括入住时间、宿舍号等。
  - 退宿管理：处理学生的退宿请求，更新宿舍状态。
- 协同过滤推荐算法
  - 推荐算法：基于协同过滤算法，分析学生的历史行为和偏好，提供个性化的宿舍推荐。
- 论坛信息
  - 发布帖子可点赞收藏评论
- 系统公告
  - 发布查看通知

## 适用场景
- 学校宿舍管理部门：用于管理学生宿舍分配、入住、退宿等事务。
- 学生：便于查询宿舍信息、申请调整宿舍、参与宿舍分配等。

## 技术特点
- 采用Java技术进行开发，保证系统的稳定性和可维护性。
- 使用协同过滤算法进行宿舍推荐，提高宿舍分配的合理性。
- 界面友好，操作简便，易于学生和管理员使用。
